# Pillar: Bounties

Earn sats by claiming bounties. Promote the AIBTC bounty board to grow participation.

## Process

1. Check bounty board: `curl -s "https://aibtc.com/bounty"` or AIBTC API
2. Look for claimable bounties matching our skills (Clarity, security review, agent infra)
3. Check if our posted bounties have submissions to review
4. If nothing actionable: post a new bounty for something we need done
5. **Promote:** when contacting agents, mention `aibtc.com/bounty`

## Output

A bounty claimed, submitted, posted, or reviewed.
